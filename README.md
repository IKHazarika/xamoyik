# উদ্দেশ্য আৰু যুক্তি

এই বটে অসমীয়া ৱিকিপিডিয়াৰ সেই প্ৰবন্ধসমূহ বিচৰাৰ চেষ্টা কৰে যাক সাময়িক সংশোধনৰ প্ৰয়োজন থাকিব পাৰে। তেনে কৰিবলৈ এই বটে নিম্নলিখিত যুক্তি প্ৰয়োগ কৰে:

১। যদি অসমীয়া ৱিকিপিডিয়াত সেই প্ৰবন্ধটো "জীৱিত ব্যক্তি", "দেশ" আদি শ্ৰেণীত ৰখা হৈছে

২। অন্যথা যদি Wikidata জীৱিত ব্যক্তি, সাময়িক ঘটনা আদি item এই প্ৰবন্ধৰ সৈতে যুক্ত হৈ আছে

৩। অন্যথা যদি প্ৰবন্ধটোৰ ইংৰাজী ৱিকিপিডিয়াৰ সংস্কৰণত "জীৱিত ব্যক্তি", "দেশ" আদি শ্ৰেণী বা "কোম্পানী", "চৰকাৰী সংস্থা" আদি তথ্যবাকচ ৱয়ৱহাৰ কৰা হৈছে

৪। অন্যথা যদি প্ৰবন্ধটোৰ অসমীয়া বা ইংৰাজী সংস্কৰণত "জনসংখ্যা", "প্ৰতি ব্যক্তি আয়" আদি সংজ্ঞাৰ ব্যৱহাৰ কৰা হৈছে।

# দুৰ্বলতা

-- কিছু কিছু নিৰ্ণয় চৰ্তৰ সঠিকতা উচ্চ-মানৰ: যেনে অসমীয়া ৱিকিপিডিয়াত "জীৱিত ব্যক্তি" শ্ৰেণী সনাক্ত হ'লে বা ইংৰাজী ৱিকিপিডিয়াত {{Ongoing....}}এৰে আৰম্ভ হোৱা তথ্যবাকচৰ অস্তিত্ব। 

-- পিচে, আন কোনো কোনো চৰ্তই মাজে মাজে ভুল নিৰ্ণয় দিব পাৰে: যেনে "চৰকাৰী সংস্থা" বা "দেশ" এনে প্ৰবন্ধতো দেখা যাব যি অস্তিত্বত আছিল কিন্তু বৰ্তমানে নাই। পিচে, এই চৰ্তসমূহ এই বিশ্বাসেৰে অন্তৰ্ভুক্ত কৰা হৈছে যে অস্তিত্বত থকা সংস্থা, দেশ, ৰাজনৈতিক দল আদিৰ ওপৰত লিখা প্ৰবন্ধৰ সংখ্যা, সমস্যাযুক্ত প্ৰবন্ধতকৈ বহু অধিক। সেয়ে, এনে প্ৰবন্ধক বাদ দিলে যি ত্ৰুতি হ'ব, তাতকৈ কেইটামান false positive সহ্য কৰাই ভাল। পাছলৈ মানৱ-সম্পাদকে false positiveকেইটা আঁতৰাই দিব পাৰে।

-- আন হাতে, এনে কিছু প্ৰবন্ধৰ প্ৰকাৰ আছে যাক সাময়িক সংশোধনৰ প্ৰয়োজন আছে, কিন্তু বটটোৱে বিচাৰি নাপায়।

-- শেহতীয়া সম্পাদনাৰ তাৰিখ ব্যৱহাৰ কৰিলে কিছু বিভ্ৰান্তিৰ সৃষ্টি হ'ব পাৰে, যিহেতু হয়টো শেহতীয়া সম্পাদনা কিছু সৰু-সুৰা বানান আদি ঠিক কৰিবলৈহে কৰা হৈছিল। নতুন প্ৰবন্ধৰ বাবে ই বিশেষ সমস্যাৰ সৃষ্টি নকৰে। পুৰণি প্ৰবন্ধৰ বাবে ই এক সমস্যা। এটি আংশিক উপায় এনে ধৰণৰ হ'ব পাৰে: অন্ততঃ x বাইটৰ সম্পাদনাকহে গণ্য কৰা হ'ব, যিহেতু ডাঙৰ সম্পাদনাই কেইবাটাও তথ্য update কৰাৰ সম্ভাৱনা অধিক। পিচে যদি কোনো প্ৰবন্ধত কেৱল জনসংখ্যা, আয় আদি সংখ্যাহে update-যোগ্য, এই যুক্তি নাখাটিব। সেয়ে, এটি সংৰক্ষণশীল অনুমান হিচাপে শেহতীয়া সম্পাদনাৰ তাৰিখকে ব্যৱহাৰ কৰা হৈছে।

# পৰীক্ষা

যোৱা দুদিনত মই মোৰ একাউণ্টতে সৃষ্ট বটেৰে এই ক'ড চলাই চাইছোঁ, কেইটামান ভুলৰ বাদে প্ৰায় ঠিকেই সাঁচ যোগ সম্পন্ন হৈছে।

# About the folder structure

The python script also expects two files called user-password.cfg that includes login details for the bot, and user-config.py that specifies the Wikimedia family, language and main account details.

They have NOT been included here for security.

The format looks like this:

## user-config.py

#-*- coding: utf-8 -*-

from collections import defaultdict

family = "wikipedia"
mylang = "as"

#Pywikibot v10 expects a defaultdict
password_file = "user-password.cfg"
usernames = defaultdict(dict)
usernames["wikipedia"]["as"] = "main user name"
usernames["wikipedia"]["en"] = "main user name"
usernames["wikipedia"]["wikidata"] = "main user name"

put_throttle = 5
minthrottle = 0

cosmetic_changes = False

console_encoding = "utf-8"

## user-password.cfg

('as', 'wikipedia', 'main user name', BotPassword('bot name', 'bot secret'))
